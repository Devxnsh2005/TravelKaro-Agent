# ✈️ TravelKaro — AI-Powered Travel Planner



---

## 🌍 About the Project

**TravelKaro** is an AI-powered travel-planning web application designed to make vacation decisions easier through natural voice conversations.

Users can start a conversation with the AI travel assistant, discuss their travel preferences, and receive assistance while planning their trip.

🔗 **Live Website:** [TravelKaro Live](https://travel-karo-agent-devansh25.vercel.app)

---

## Features

* Voice-based interaction with an AI travel assistant
* Helps users begin planning their vacations through conversation
* Microphone-permission prompt for seamless voice interaction
* Start and end conversation controls
* Live status updates such as listening, speaking, and connecting
* Responsive user interface for desktop and mobile devices
* Clean, modern design with gradients and animated buttons
